# Loopr Website

Home-grown static site for the Loopr marketing + docs website.

## Requirements
- Node.js 18+

## Install
```bash
npm install
```

## Development
```bash
npm run dev
```

## Build
```bash
npm run build
```

## Preview
```bash
npm run preview
```

## Tests
```bash
npm test
```

## Cloudflare Web Analytics
Set the token via environment variable or config:
- `CF_ANALYTICS_TOKEN=<token>` or `CLOUDFLARE_ANALYTICS_TOKEN=<token>`
- Or set `analytics.cloudflareToken` in `site.config.json`
