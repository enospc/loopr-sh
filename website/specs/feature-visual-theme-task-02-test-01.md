# Test: Shared section layouts and footer

## Test ID
01

## Type
Manual

## Purpose
Ensure shared section layouts, CTA bands, and the footer render consistently.

## Preconditions
- Site built with `npm run build`.

## Test Data
- None.

## Steps
1. Open the home page and verify hero, feature sections, trust/enterprise blocks, and closing CTA band.
2. Open a docs page and confirm section spacing matches the home page rhythm.
3. Verify footer uses a multi-column layout.

## Expected Results
- Shared layout patterns render consistently across pages.

## Automation Notes
- Can be automated by checking for section class names and footer markup.
