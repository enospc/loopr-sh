# Feature: Codex power user narrative

## Summary
Add a dedicated section that teaches developers how to become Codex power users and adopt AI-first software engineering, grounded in Loopr.md.

## Goals
- Provide actionable guidance for AI-first engineering
- Connect Loopr workflow to Codex power usage
- Reflect Loopr.md’s mindset and reliability principles

## Non-goals
- Full training curriculum or certification
- Copying Loopr.md verbatim

## User Stories
- As a developer, I want a clear path to becoming an AI-first engineer.
- As a team lead, I want guidance to help my team adopt AI workflows safely.

## Scope
- In scope:
  - Narrative page explaining AI-first engineering
  - Practical tips for Codex power usage
  - How Loopr supports disciplined workflows
- Out of scope:
  - Video courses or long-form curriculum

## Requirements
- Content should be concise, actionable, and linked from nav.
- Provide links to relevant docs sections.
- Content grounded in Loopr.md’s principles (verification, outcomes, short loops).

## Acceptance Criteria
- Dedicated page exists and is linked from nav.
- Content aligns with Loopr workflow and Loopr.md principles.

## UX / Flow
- Standalone page with sections and quick links to docs.

## Data / API Impact
- None.
