# Test: Design tokens and typography applied

## Test ID
01

## Type
Manual

## Purpose
Verify global design tokens and typography are applied consistently.

## Preconditions
- Site built with `npm run build`.

## Test Data
- None.

## Steps
1. Open the home page and a docs page in the build output.
2. Verify that headings, body text, and buttons share the same typography scale.
3. Confirm the color palette, background, and spacing feel consistent across pages.

## Expected Results
- Global theme appears consistent across marketing and docs pages.

## Automation Notes
- Can be partially automated by checking for CSS variable usage.
