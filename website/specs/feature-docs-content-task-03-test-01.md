# Test: FAQ page content and Loopr.md alignment

## Test ID
01

## Type
Manual

## Purpose
Verify the FAQ page exists, includes key topics, and reflects Loopr.md principles.

## Preconditions
- Site built with `npm run build`.
- Loopr.md available for reference.

## Test Data
- None.

## Steps
1. Open the FAQ page in the build output.
2. Confirm it includes questions about installation, safety, and workflow.
3. Verify links from FAQ answers to relevant docs sections.
4. Spot-check content for alignment with Loopr.md principles.

## Expected Results
- FAQ page exists with required topics and valid links.
- Content aligns with Loopr.md principles.
